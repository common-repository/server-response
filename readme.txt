== Description ==

Server Response поможет вам скорректировать заголовки ответа сервера и отключить REST API.

Функции плагина:

1. Отключение REST API.

1. Удаление ссылки на REST API из ответа сервера.

1. Актуализация даты заголовков Expires и Last-Modified в ответе сервера. Создание данных заголовков, если они отсутствуют.

1. Удаление rel=shortlink из ответа сервера.


== Installation ==

1. Разархивируйте файлы плагина.
2. Загрузите папку в директорию `/wp-content/plugins/`.
3. Активируйте плагин через меню `Плагины`.
